# 人脸识别系统简介 python270

## 项目概述

人脸识别系统是基于深度学习技术实现的，具有高精度的人脸检测与识别能力。系统可完成人脸的识别、录入和管理等多项功能，支持多种数据源输入，并提供实时的人脸识别服务。

## 技术栈

- 开发语言：Python 3.8
- 开发工具：PyCharm
- 依赖管理：requirements.txt
- UI框架：Qt（.ui、.qrc）

## 功能模块

### 人脸识别

- 实时识别已录入的人脸
  - 支持图片、视频、摄像头输入
- 快速检测与识别

### 人脸录入

- 通过图片和摄像头检测并录入新的人脸数据

### 人脸管理

- 系统管理和更新人脸数据

## 系统角色

- 用户：使用系统进行人脸的录入与识别
- 管理员：管理系统中的人脸数据

## 运行环境

- Python版本：3.8
- 所需依赖包：详见表requirements.txt

## 部署步骤

1. 确保Python 3.8环境已搭建
2. 安装依赖包，执行 `pip install -r requirements.txt`
3. 运行主程序 `runMain.py` 启动界面
4. 使用 `testFaceDemo.py` 进行测试图片识别
5. 使用 `testGetFeatures.py` 进行人脸特征提取测试
